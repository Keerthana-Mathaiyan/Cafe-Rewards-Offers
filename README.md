# ☕ Cafe Sales Analytics Platform

A Business Intelligence dashboard built using **Streamlit**, **MySQL**, **Pandas**, and **Plotly** to analyze cafe customer behavior, offer performance, and engagement insights.

---

# 📌 Project Overview

This project provides an interactive analytics platform for cafe sales and marketing analysis.

The dashboard helps businesses:

- Understand customer demographics
- Analyze customer engagement
- Evaluate offer effectiveness
- Track marketing channel performance
- Generate business insights visually

---

# 🚀 Features

## 📊 Project Insights
- Executive KPI dashboard
- Offer funnel analysis
- Event distribution tracking
- Customer engagement analysis
- Business recommendation insights

## 👤 Customer View
- Age distribution analysis
